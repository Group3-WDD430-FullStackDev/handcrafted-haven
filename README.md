## Getting Started
#### Install dependencies
```bash
npm i
# or
npm install
```

#### Run the development server:
```bash
npm run dev
```

Open ([http://localhost:3000](http://localhost:3000)) with your browser to see the result.

#### Format and Lint Code
```bash
# format
npm run format 

# lint
npm run lint

# run both
npm run prepare
```
